חיבור פרסום לאתר

כדי שכפתור "פרסם" יעדכן את המסך לבד, צריך פעם אחת טוקן (סיסמה אישית) של GitHub. הוא נשמר רק בדפדפן הזה ולא נשלח לאף אחד.

  1. פתח: יצירת טוקן ב-GitHub ↗
  2. שם כלשהו (למשל "margesh"), תוקף — לבחירתך.
  3. Repository access → "Only select repositories" → בחר margesh-screen.
  4. Permissions → Contents → Read and write.
  5. לחץ Generate token, העתק והדבק כאן 👇
